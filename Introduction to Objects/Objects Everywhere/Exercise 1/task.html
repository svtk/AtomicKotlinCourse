Objects Everywhere (#1)

Palindromes are words or phrases that read the same forward and backward. Some examples include "mom" and "dad." Write a function to test words or phrases for palindromes.

Some examples: "mom" is a palindrome "dad" is a palindrome "street" is not a palindrome

`String`'s `reversed` member function may prove useful here. Use `==` to check whether two strings are equal.
