Mastering the IDE: Refactor this

You can extract a value used inside a function to a parameter. To do that you can use the Extract parameter refactoring. Alternatively, call Refactor this action by pressing &shortcut:Refactorings.QuickListPopupAction;. Then you may choose Extract parameter by pressing the corresponding number on the keyboard.

Refactor this is a helpful way to check which ways to refactor code are available.
