Mastering the IDE: Navigating to a Function

You can easily see how a function is defined. To navigate to a function definition, put the caret on the usage and press &shortcut:GotoDeclaration;. Alternatively, you may choose Navigate in the application menu, and then Declaration.

Put the caret on the oneOrTheOther() function usage in main() and navigate to the function definition. Your caret should move to the oneOrTheOther() function definition.
