## Summary 1 (#2)

Reverse a decimal number using a while loop. An auxiliary `worker` variable
will help you iteratively produce the `result`:

| Worker     | Result |
|------------|--------|
| 1234       | 0      |
| 123        | 4      |
| 12         | 43     |
| 1          | 432    |
| 0          | 4321   |
