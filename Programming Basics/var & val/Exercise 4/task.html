Mastering the IDE: Project Tool Window

The icons at the top of the Project Tool Window allow to perform several convenient operations. For example, you can locate the currently opened file in the Project Tool Window by clicking on the Scroll from Source icon.

Alternatively, press &shortcut:SelectIn; or choose Navigate | Select in... from the menu above. Then choose Project View in the open menu.

Press the Collapse All button on the Project Tool Window to hide all the project content, then open the current file by trying all the options listed above.
