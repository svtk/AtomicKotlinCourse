Mastering the IDE: Find Action

Memorizing the shortcuts for the different actions takes time. After a while, you'll learn automatically the ones that you use often, but remembering all of them is almost impossible.

To solve this, IntelliJ IDEA provides a universal way to call an action by its name. Simply press &shortcut:GotoAction;. Alternatively, you may choose Help | Find Action on the main menu.

Find action allows you to enter the name you're looking for. You may type only a part of the name, and IntelliJ shows all the names that fit.

Try typing the names of actions that we've seen so far:

  • Specify type explicitly;
  • Rename;
  • Convert to expression/block body;
  • Navigate to declaration.

You can display the intention actions by calling the action Show intention actions, but it's such a common operation that it's better to memorize a shortcut for it.
